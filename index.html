About StudyMind AI

StudyMind AI is an offline-first, client-side personal AI study concierge built to help students schedule tasks, study efficiently with Pomodoros, check in on energy level moods, and learn actively.

This project was built for the Google x Kaggle Capstone Project competition, specifically under the Concierge Agents Track. It showcases how a small-scale agent can assist in day-to-day productivity using modern client-side integrations.

Key Concepts Demonstrated:

  • Zero-Cost / Serverless Architecture: Uses direct Gemini API calls from the browser with the student's personal key.
  • Data Privacy: All task boards, Pomodoro intervals, streaks, and API keys reside completely in local browser storage.
  • Mood-Driven Advice: Integrates a mood check-in logger (1-5 energy levels) to supply context to the LLM agent, optimizing the suggestions.
  • Integrated Tools: Dynamic timeline creation, quiz construction, and raw syllabus parsing into manageable structural tasks.

Gemini API Key details:

You can get a free, high-limit key (supporting gemini-2.5-flash-lite) directly from Google AI Studio.
